# Каталог ресурсов

## Быстрые действия

- [[Resources/templates/Ресурс - шаблон|Добавить ресурс (открыть шаблон)]]
- **Новые ресурсы**  
  - Представление Bases: *Resources / Все ресурсы / status = new*
- **Без категории**  
  - Представление Bases или Dataview: ресурсы без `category`
- **Без ссылки (source_url)**  
  - Представление Bases или Dataview
- **Готово к выгрузке**  
  - Представление Bases: *status = uploaded* или другое финальное значение
