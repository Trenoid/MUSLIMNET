# Программирование

## Описание
Категория ресурсов по программированию, разработке и смежным темам.

## Ресурсы этой категории

```dataview
TABLE
  file.link AS "Ресурс",
  type,
  subcategory_1,
  subcategory_2,
  language,
  status,
  source_url,
  updated
FROM "Resources/items"
WHERE contains(tags, "resource")
  AND category = [[Программирование]]
SORT updated DESC
```

## По типам

```dataview
TABLE
  file.link AS "Ресурс",
  type,
  subcategory_1,
  subcategory_2,
  status
FROM "Resources/items"
WHERE contains(tags, "resource")
  AND category = [[Программирование]]
GROUP BY type
SORT updated DESC
```
