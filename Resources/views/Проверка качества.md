# Проверка качества

## Без типа

```dataview
TABLE title, category, source_url, status
FROM "Resources/items"
WHERE contains(tags, "resource")
  AND (type = null OR type = "")
```

## Без категории

```dataview
TABLE title, type, source_url, status
FROM "Resources/items"
WHERE contains(tags, "resource")
  AND (category = null OR category = "")
```

## Без source_url

```dataview
TABLE title, type, category, status
FROM "Resources/items"
WHERE contains(tags, "resource")
  AND (source_url = null OR source_url = "")
```

## Дубликаты по названию

```dataview
TABLE rows.file.link AS "Файлы"
GROUP BY title
WHERE contains(tags, "resource")
  AND length(rows) > 1
```

## Со статусом new

```dataview
TABLE title, type, category, source_url, updated
FROM "Resources/items"
WHERE contains(tags, "resource")
  AND status = "new"
SORT updated ASC
```

## Давно не обновлялись (например, > 180 дней)

```dataview
TABLE title, type, category, updated
FROM "Resources/items"
WHERE contains(tags, "resource")
  AND updated <= date(today) - dur(180 days)
SORT updated ASC
```
