Описание: 


Цель сделать большую экосистему программ, которая будет развернута в своей сети. Будут программы по типу ютуб, телеграмм, и так далее.  Сделать для них всех один общий авторизационный сервис. То есть будет один аккаунт для всех программ. Продумать что должен уметь этот централизованный авторизационный  сервис, какие возможности у него должны быть. Он должен уметь запрещать использование одних программ и запрещать другие, должна быть некая система прав для каких-то действий. Например что бы смотреть видеохостинг права не нужны, а чтобы создать аккаунт который может выкладывать видео, нужны. И так далее




Необходимые программы:
1. Аналог ютуб
2. Аналог стриминговых платформ
3. Аналог телеграмм
4. Локально развернутые ИИ
5. Онлайн Библиотека.
6.  Гитхаб или гитлаб локально
